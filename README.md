# 🛡 AI Toxic Comment Detection System

A Machine Learning + Streamlit web application that detects whether a comment is **Toxic or Safe** using Natural Language Processing (NLP).

---

## 🚀 Live Demo
Try the app here 👉  
https://toxic-data-dashboard.streamlit.app/

---

## 📌 Features
- 🔍 Real-time toxic comment detection  
- 🤖 Machine Learning based classification  
- 📊 Confidence score visualization  
- 👤 Admin & Guest login system  
- 📜 Prediction history tracking  
- 📈 Analytics dashboard  

---

## 🧠 Tech Stack
Python, Streamlit, Pandas, Scikit-learn, Matplotlib, NLP (TF-IDF)

---

## ▶️ Run Locally
streamlit run app/app.py
